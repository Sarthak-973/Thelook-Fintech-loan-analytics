# 📊 Unlocking Loan Insights | Cloud Data Analytics

**Prepared by:** Sarthak D  
**Project Type:** Google Cloud Data Analytics Capstone  

---

## 🔹 Overview
This project was completed as part of the Google Cloud Data Analytics Capstone. The goal was to help the Treasury department at TheLook Fintech understand important metrics such as cash flow, loan purpose, and borrower location.

As part of this project, I created a report and an interactive dashboard using Looker. The analysis focused on the total number of loans issued and key metrics to track loan performance and support better decision-making.

---

## 🎯 Objectives

- Analyze the Fintech dataset to understand loan trends, including the total number of loans issued by day and year  
- Build an interactive dashboard to help the Treasury team easily track important loan performance metrics  

---

## 🛠️ Tools & Technologies

- Google BigQuery (SQL for data processing and analysis)  
- Looker Enterprise (Dashboard creation and visualization)  
- Google Cloud Platform (Cloud environment)  
- SQL (CTAS, JOIN, GROUP BY, DISTINCT)  

---

## 📊 Results

- The analysis shows that the total number of loans increased steadily from 2012 to 2019  
- Created a table (`loan_count_by_year`) to track yearly loan trends  
- Worked with key metrics such as:
  - Loan count over time  
  - Loan purpose  
  - Borrower location  

---

## 📈 Dashboard Insights

The Looker dashboard includes:
- Total Outstanding Loan Amount (KPI)  
- Loan Status Distribution (Pie Chart)  
- Top States by Loan Count (Bar Chart)  
- Top Customers by Income (Table)
